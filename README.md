# Laiba Badar 🍽️

Open-source **food brand + delivery** stack for [laibabadar.com](https://laibabadar.com).

## Monorepo

| Path | What |
|------|------|
| `apps/web` | Node site / API server |
| `apps/android` | Kotlin delivery app |
| `apps/brand-site` | Marketing / brand pages |

## Download & use

```bash
git clone https://github.com/badrpk/laibabadar.git
cd laibabadar

# Web
cd apps/web && npm install && npm start

# Android — open apps/android in Android Studio
```
